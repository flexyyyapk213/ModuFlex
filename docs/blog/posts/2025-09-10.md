# ModuFlex v0.0.9b1

- Новые возможности для разработчиков.
- Добавлен новый пак: AIFuncs.Ознакомиться: .help AIFuncs
- Если у вас не работает команда .ping, в терминале напишите 'pip uninstall speedtest' и 'pip install speedtest-cli'.
- Добавлен 'плагин' ModuFlex, информация о всех системных командах.
- Теперь бот запускается со своим venv, что улучшает стабильность и избегает конфликтов с другими библиотеками.
- (Для разработчиков)Добавлена возможность хранить конфигурацию в общем файле при помощи Data(подробнее в contribution.md).
