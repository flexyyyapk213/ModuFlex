# Информация

Здесь собран список версий ModuFlex, в котором описано нововведения/изменения.

Каждая версия относиться к категориям Release/Beta/Alpha

Про категории:

!!! success "Release"

    Стабильная версия, обычно не содержит критических багов.

!!! warning "Beta"

    Нестабильная версия, создана для тестировании новых возможностей, ожидаются баги разных видов.

!!! example "Alpha"

    Древняя версия, зачаток начала ModuFlex, включала в себя самые базовые функции и могла быть нестабильной. Использовалась для первоначального тестирования архитектуры и идей, часто содержала множество багов и служила основой для всех последующих улучшений, обретения опыта и новых знаний.Alpha версии считаются 0.0.X
